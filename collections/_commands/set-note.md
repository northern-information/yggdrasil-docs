name: Set Note
signatures: ["_X_ _Y_ _MIDI_NOTE_", "_X_ _Y_ _YGG_NOTE_",]
examples: ["4 3 72", "4 3 c5"]
category: musical
description: Set a slot's MIDI value. Also accepts "ygg" notation which is like [IPN](https://en.wikipedia.org/wiki/Scientific_pitch_notation) but instead of the `#` character simply use a lower case `s`.
